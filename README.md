# NeuroBotics Dashboard - Actividad 2 Grupal (Grupo 10) - UNIR

Panel de control interactivo desarrollado para la actividad 2 de Tecnologias Emergentes impartida por Pablo Antonio Gargallo Jaquotot  , construido con **React JS**, **Tailwind CSS** y **Shadcn UI**. El estilo visual está fuertemente inspirado en el efecto *Liquid Glass* de Apple y en referencias modernas de diseño como este [AI Dashboard](https://dribbble.com/shots/27328098-AI-Greenhouse-Management-Dashboard).
